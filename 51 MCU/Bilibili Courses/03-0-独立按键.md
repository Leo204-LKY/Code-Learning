### 3-1-独立按键控制 LED 亮灭  
上章中如果需要控制某一 LED 的亮灭，需要同时控制 8 个 LED  
```C
P2 = 0xFE;
```
实际上 `<REGX52.H>` 中有对每个引脚进行定义，可以进行只改变其中一个  
如果需要控制 D1 LED 灯，只需将 `P2` 改为 `P2_0`  
```C
P2_0 = 1;
```
使用 `if` 检测独立按键 K1 是否被按下，然后控制灯的亮灭  
```C
#include <REGX52.H>

void main()
{
    while(1)
    {
        if(P3_1 == 0)  // 注：当 P3_1 的值为 0 (低电平) 时按钮被按下
        {
            P2_0 = 0;  // 灯亮
        }
        else
        {
            P2_0 = 1;  // 灯灭
        }
    }
}
```
