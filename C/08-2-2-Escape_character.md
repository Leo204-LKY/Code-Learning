# 逃逸字符  
用于表达无法印出来的控制字符或特殊字符，由一个反斜杠 `\` 开头，后面跟上另一个字符，这两个合起来组成一个字符  
之前求身高相关的代码里出现了 `\"`
```C
printf("请分别输入身高的尺寸和英寸，"
    "如输入\"5 7\"表示 5 英尺 7 英寸：");
```

## 逃逸字符的种类  
| 字符 | 意义 | 字符 | 意义 |
| :---: | :---: | :---: | :---: |
| `\b` | 回退一格*1 | `\"` | 双引号 |
| `\t` | 到下一个表格位*2 | `\'` | 单引号 |
| `\n` | 换行*3 | `\\` | 反斜杠本身 |
| `\r` | 回车*3 |  |  |

- ***1** - 回退一格不等于退格键删除  
- ***2** - 制表位  
    - 每一行的固定位置  
    - 一个 `\t` 使得输出从下一个制表位开始  
    - 用 `\t` 可以使上下两行对齐，如：  
        ```C
        printf("123\t456\n");
        printf("12\t456\n");
        ```
        ```
        123 456
        12  456
        ```
- ***3** - 回车与换行  
    - 源自打字机的动作  
        - 回车是将打字头从纸的右侧移到左侧  
        - 换行是旋转纸筒来使打字头移向新的一行  
    - 反映在 C 语言中， `\t` 和 `\n` 也分别指回车和换行  
        - 但终端会自动将 `\n` 解释为回车且换行
